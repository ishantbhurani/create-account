## Run Locally

Clone the project

```bash
  git clone git@github.com:ishantbhurani/create-account.git
```

Go to the project directory

```bash
  cd create-account
```

Run `index.html`

```bash
  <browsername> index.html
```

E.g.

```bash
  firefox index.html
```

```bash
  google-chrome index.html
```
